# Contact-Manager
Contact Management System (Java)

The Contact Management System is a simple, user-friendly Java console application designed to help users manage personal contact information efficiently. It allows users to store, view, update, and delete contact details such as name, phone number, and email address through a clear, menu-driven interface.

Features

Add new contacts with basic details

View all saved contacts in a structured format

Edit existing contact information

Delete contacts that are no longer needed

Exit the application safely

Technologies Used

Java programming language

ArrayList for dynamic data storage

Scanner class for user input

Key Concepts Covered

Object-Oriented Programming principles

Use of classes and objects

Working with Java collections

Conditional statements and loops

Menu-driven application design

Use Case

This project is well suited for Java beginners who want hands-on practice with CRUD operations. It helps build a strong foundation in OOP concepts and improves confidence in developing console-based applications.
